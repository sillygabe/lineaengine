# Welcome to Linea

Linea is a little (and silly) game engine for C++. It prioritizes developer experience and linear approach for learning.
It is based on *awesome* libraries like `SDL` and `OpenGL` (`GLFW`).

# QUICKSTART

REQUIREMENTS:
1. SDL
2. GLFW (Not now, but soon)

Download these libraries **first**

Then download the release for your platform, unpack it, and do what the instructions tell you!
